appointments: Должности участников (у каждого участника их может быть несколько)

id Ай ди челика
appointment Должность

basic_logic: Вопросы для теста на базовую логику

id Ай ди вопроса
text Текст вопроса
right Правильный ответ
wrong_1 Неправильный ответ
wrong_2 Другой неправильный ответ

basic_logic_tested: Кто как прошёл тест на базовую логику

id Ай ди проверяемого
question_1 Первый вопрос, который был задан (после идут вопросы 2 -- 6)
...
answer_1 Первый ответ, который был дан (после идут ответы 2 -- 6)
...
time Время, когда чел сдал тест (на данный момент не использующаяся)

channels: Каналы

id Ай ди канала
name Простое название канала, типа "Недостримы" "Голосовашки" "Проколы"

chats: Чаты

id Ай ди чата
name Название чата
purpose Его предназначение (Главный чат, Чат, Админосостав, Комитет)
type: Тип (не используется)
link: Ссылка (не используется)
standart_commands: Работают ли стандартные команды в чате (не используется)
boss_commands: Команды более мощных челиков (не используется)
financial_commands: денежные команды (не используется)
mutual_invites: обмен ссылками (не используется)
messages_count: сообщения считаются, как в системе МФ2 (не используется)
violators_ban: автобан нарушителей (не используется)
admins_promote: автоадминки админам (не используется)

config: Некоторые дополнительные переменные, значения которых нужно хранить

var Название переменной
value Её значение

cooldown: Кулдауны команд

person id Ай ди юзера команды
command команда
chat id Ай ди чата, где команда использована
time Время в секундах, когда команда была последний раз выполнена

elite_results: Не использующаяся таблица

id Ай ди чела
basic_logic_results Количество баллов в тесте на базовую логику

languages: Настройки языков

id: ай ди чата, при чём как лички, так и чаты на несколько человек
language: Включённый там язык

members: Участники

id Ай ди чела
username Его юзер
nickname Его ник
rank Его звание, вроде Гость или Член Комитета
messages Количество сообщений, отправленных им
warns Количество полученных им варнов
money Количество ЯМ на его счету
month_birthday месяц рождения челика
day_birthday день рождения челика

messages: Сообщения

person_id Ай ди челика
chat_id: Ай ди чата
message: Количество сообщений
